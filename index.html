about me


לאה גולדברג נולדה ב-03.02.1950 בפולין בוורצלב להורים מאשה ושלמה
נשואה לזאב גולדברג ולהם 2 ילדים ו-5 נכדים
גדלה בפולין עד גיל 7, למדה ריקוד ומשחק והופיעה בהצגות
למדה בכיתה א בפולין וכעבור חצי שנה עלתה ארצה ברכבת לאוסטריה, משם לאיטליה ובהפלגה באוניה לחיפה
בצעירותה לקחה חלק בנוער העובד והלומד ורקדה ריקודי עם
את זאב הכירה כשגרה בלוד כשהייתה בת 15 והוא בן 17, לאחר חברות ממושכת התחתנו ב-01.06.69
למדה בבית ספר למטפלות מוסמכות בויצו בתל אביב ונשארה לעבוד במחלקת פגים בבית החולים אסף הרופא השילוב של עבודה במשמרות ושני ילדים קטנים הובילו אותה לשנות מקצוע, ללמוד בתל אביב חינוך לגיל הרך ולעבוד במעונות נעמת
כיום אוהבת להיות עם משפחתה, לטייל ולבשל

Q & A


מהם ההבדלים העיקריים בין web 1.0, web 2.0 ו-web 3.0?
1. Web 1.0 מכיל תוכן לקריאה בלבד read only המידע הוא סטטי כאשר המטרה העיקרית היא להשיג מידע ואין אינטראקציה בין הקוראים למפרסמי האתרים, web 2.0 מאפשר קריאה וכתיבה ובכך מתאפשר שיתוף מידע, דינאמיות ואינטראקציות ברשתות חברתיות, דפי ויקיפדיה. Web 3.0 נחשב read, write and execute המידע מותאם אישית למשתמש.

2. ב-web 1.0 אין טכנולוגיה המאפשרת מנוע חיפוש, ב-web 2.0 קיים מנוע חיפוש על פי תוכן וב-web 3.0 מעבר ל-AI, עיבוד שפה טבעית, קיים אינדוקס.

3. Web 1.0 הוא ברובו צד לקוח, web 2.0 מתמקד בצד שרת, יש שימוש בדאטה בייסים וב-web 3.0 יש קישור בין שרתים, שימוש בבינה מלאכותית, טכנולוגיות cloud ו-IOT.

חיפוש מידע
ב-web 1.0 המשתמש מחפש אתר רלוונטי בפורטל. התנאי המקדים – המשתמש צריך לדעת את ה-URL של הדף כדי להגיע אליו.
ב-web 2.0 המשתמש מחפש שאילתא מסוימת בגוגל, המערכת מחזירה למשתמש את כל התוכן שקיים בנושא.
ב-web 3.0 המשתמש מחפש שאילתא מסוימת בגוגל, המערכת מחזירה למשתמש את התוצאות הטובות ביותר הקיימות על פי דירוג.

דינאמיות התוכן
ב-web 1.0 החברה אשר בבעלותה האתר שומרת את קובץ האתר פיזית על מנת לבצע עדכון.
ב-web 2.0 משתמש מעדכן תוכן בפלטפורמות מסוימות.
ב-web 3.0 משתמש בונה יישומים שאנשים אחרים יכולים ליצור איתם אינטראקציה. חברות בונות פלטפורמות המאפשרות לאנשים לפרסם שירותים על ידי מינוף הקשרים בין אנשים או תוכן מיוחד.
